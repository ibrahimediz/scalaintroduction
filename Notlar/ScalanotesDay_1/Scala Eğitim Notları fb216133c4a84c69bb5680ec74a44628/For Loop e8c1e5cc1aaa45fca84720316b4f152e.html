For Loop

  • Ön Bilgi

    Scala 3 ile birlikte, for döngüsünün sözdizimi de değişti. Artık daha basitleştirilmiş bir yapı sunulmuştur. İşte Scala 3'te kullanılan yeni for döngüsü örneği:

    // Temel kullanım
for (i <- 1 to 5) {
  println(s"Sayı: $i")
}

// Adım belirterek kullanım
for (i <- 1 to 10 by 2) {
  println(s"Tek Sayı: $i")
}

// Dizi üzerinde gezinme
val isimler = Array("Ayşe", "Fatma", "Cemil")
for (isim <- isimler) {
  println(s"İsim: $isim")
}

// Tuple kullanımı
val koordinatlar = List((1, 2), (3, 4), (5, 6))
for ((x, y) <- koordinatlar) {
  println(s"Koordinat: ($x, $y)")
}

    Bu örneklerde, for döngüsü oldukça basitleştirilmiş bir şekilde kullanılmıştır. Scala 3'te for döngüsü, for (değişken <- koleksiyon) şeklinde daha açık ve okunabilir hale gelmiştir. Ayrıca, adım belirterek veya dizi üzerinde gezinmek gibi durumlar için de rahat bir kullanım sağlanmıştır.

    Bu yeni sözdizimi, Scala'nın daha fonksiyonel ve ifade odaklı bir dil olma özelliklerini korur.

  • Detay

    Scala 3'te for döngüleri, bir dizi veya koleksiyondaki her öğe üzerinde yinelemek için kullanılan güçlü bir araçtır. Farklı türde for döngüleri mevcuttur, her biri farklı bir kullanım sunar.

    Temel For Döngüsü Örneği

    val sayilar = Seq(1, 2, 3)
for i <- sayilar do println(i)

// ya da

for (i <- 1 to 10) {
  println(i)
}

// ya da 

for i <- sayilar // sayilar listesinden referansla (generator)
do
  val x = i * 2
  println(s"i = $i, x = $x")

// ya da 

val sayilar = Seq(1,2,3)
sayilar: Seq[Int] = List(1, 2, 3)

    val sayilar = 1 to 10
// ikişer adımlı yazmak istersem

    Guards

    for döngüleri if koşulları içerebilirler bu koşullara guards denir.

    for
  i <- 1 to 5
  if i % 2 == 0 // Guard
do
  println(i)

    birden fazla guards kullanıabilir

    for
  i <- 1 to 20
  if i > 5
  if i < 8
  if i % 2 == 0
do
  println(i)

    Map ile for kullanımı

    val baskentler = Map(
"Turkiye" -> "Ankara", 
"Almanya" -> "Berlin", 
"Fransa" -> "Paris")

for (k, v) <- baskentler // k,v nin bir tuple olarak ele alındığına dikkat edelim
do
println(k + "->" + v)

    Yukarıdaki örnekte for döngüsü bir println fonksiyonu ile çıktı veriyor ancak önemli olan bir değer döndürerek bu for döngüsünün çalışmasını sağlamaktır. Bu nedenle yield deyiminin kullanımı için örneğimizi inceleyelim.

    val liste =
  for i <- 5 to 7
  yield i * 2

    Yukarıdaki örnek incelendiğinde 5 to 7 ifadesi bir vektör üretir (5,6,7) ardından yield ile döngünün her adımında değerler üretilir ve yine bir vektör olarak çıktı verir. Vector(10,12,14)

    Yukarıda yer alan örneğin kısa bir halini map fonksiyonu ile yazabilmemiz mümkündür. 

    val liste = (5 to 7).map(i => i * 2)

    for döngüsünü bir koleksiyon içerisindeki tüm elemanlara bir etki uygulamak istediğiniz zaman kullabilirsiniz. Aşağıdaki örneği inceleyelim

    val isimler = List("_ahmet","_mehmet","_ayşe")

var temizlenmisIsimler = isimler.map(x=>x.drop(1))

// ya da 
var temizlenmisIsimler2 = for isim <- isimler yield
    val temizisim = isim.drop(1)
    val basharf = temizisim.capitalize
    basharf
println(temizIsimler2) // Çıktı =>  List(Ahmet, Mehmet, Ayşe)

    For döngüsünü bir fonksiyonu içeriği olarak kullanabiliriz. Bu noktada yield deyimi oldukça kullanışlı olacaktır. 

    sayılar.filter(x=>x>=3 && x<=10)
// ya da 
for (x<-sayılar if x>=3 && x<=10) yield x
// ya da
for x<-sayilar
    if x>=3
    if x<=10
yield x
