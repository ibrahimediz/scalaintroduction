If/else if/else

  • Ön Bilgi

    Scala'da if, else, ve else if ifadeleri, koşullu ifadeleri oluşturmak için kullanılır. İşte temel kullanımı:

    val x = 10

if (x > 5) {
  println("x 5'ten büyük")
} else {
  println("x 5'e eşit veya daha küçük")
}

    Bu örnekte, x 5'ten büyükse ilk blok çalıştırılır, aksi takdirde ikinci blok çalıştırılır.

    else if Kullanımı:

    val x = 10

if x > 5 then
  println("x 5'ten büyük")
else
  println("x 5'e eşit veya daha küçük")

val y = 20

if y > 25 then
  println("y 25'ten büyük")
else if y > 15 then
  println("y 15 ile 25 arasında")
else
  println("y 15'ten küçük veya eşit")

    Bu örnekte, else if ifadesi, bir önceki if ifadesinin koşulu sağlanmazsa kontrolü sağlar. Birden çok else if bloku ekleyerek çoklu durumları kontrol edebilirsiniz.

    Tek Satırlık If-Else İfadeleri:

    Scala'da if-else ifadelerini tek satırda da kullanabilirsiniz. Ancak, bu durumda süslü parantezler gerekmez ve ifade sonucu kullanılır.

    val z = 30
val result =
  if z > 25 then "z 25'ten büyük" else "z 25'e eşit veya küçük"

println(result)

    Bu örnekte, result değişkeni, if ifadesinin sonucuna bağlı olarak atanır.

    Koşullu ifadeler Scala'da oldukça esnek ve güçlüdür. İsterseniz, match ifadesini kullanarak daha karmaşık durumları da yönetebilirsiniz.
